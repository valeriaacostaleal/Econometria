Descripción
Análisis econométrico de un portafolio de 11 acciones (NVDA como variable dependiente) usando regresión lineal, panel data, series de tiempo y GARCH.

## Archivos
- Regresiones_Proyecto_Final_COMPLETO.Rmd - Regresiones en R (diarias + intradiarias 5min)
- Regresiones_Proyecto_Final_COMPLETO.html - Output del R Markdown
- Proyecto_Final_Econometria_v2.ipynb — Panel Data + Series de Tiempo + GARCH en Python
- panel_data_enhanced.csv — Dataset del panel (11 empresas x 17 trimestres)
- DAILYPRICESALL5Y.xlsx — Precios diarios de Refinitiv (2021-2026)
- INTRADAY5MIN.xlsx — Precios intradiarios 5 min de Refinitiv
- FS___5Y.xlsx — Financial Summaries de Refinitiv (11 archivos)

## Datos
Refinitiv Workspace & Yahoo Finance

## Herramientas
R (quantmod, lmtest, car, sandwich), Python (linearmodels, statsmodels, pmdarima, arch)
